# FocusFlow — اپ اندروید

پروژه‌ی کامل Android Studio (کاتلین) که دقیقاً مثل فایل HTML خودت رفتار می‌کنه، ولی به‌صورت اپ واقعی اندروید:

- برنامه‌ی زمانی دقیقاً همون الگوریتم فایل اصلی (بلوک‌های مطالعه ۱۰۵ دقیقه‌ای + استراحت ۱۵ دقیقه‌ای ادغام‌شده، ناهار ساعت ۱۳:۰۰، شام ساعت ۲۱:۰۰).
- آلارم واقعی با `AlarmManager` — حتی اگر گوشی قفل باشه یا اپ بسته باشه، سر وقت زنگ می‌زنه و صفحه رو باز می‌کنه.
- برای خاموش کردن آلارم باید یکی از همون ۱۰۰ سؤال ریاضی بانک اصلی رو حل کنی.
- می‌تونی هر session رو فعال/غیرفعال کنی یا اسم بخش‌های مطالعه رو تغییر بدی (مثل نسخه‌ی وب، با کلیک روی اسم).
- تم تیره + زرد، شبیه دیزاین اصلی.

## چطور بسازمش (Build)

۱. **Android Studio** رو نصب کن (آخرین نسخه، رایگانه): https://developer.android.com/studio

۲. پوشه‌ی `FocusFlow` (همینی که دانلود کردی) رو با Android Studio باز کن:
   `File → Open` → پوشه‌ی FocusFlow رو انتخاب کن.

۳. صبر کن Gradle Sync تموم بشه (بار اول چند دقیقه طول می‌کشه، نیاز به اینترنت داره).

۴. گوشی اندرویدت رو با کابل USB وصل کن و **USB Debugging** رو در Developer Options فعال کن،
   یا از یک شبیه‌ساز (Emulator) داخل Android Studio استفاده کن.

۵. دکمه‌ی سبز رنگ ▶ (Run) رو بزن. اپ روی گوشی نصب و اجرا می‌شه.

## نکات مهم بعد از نصب روی گوشی واقعی

- اولین بار که اپ رو باز می‌کنی، ازت اجازه‌ی نوتیفیکیشن و "Alarms & reminders" (آلارم دقیق) می‌خواد — هر دو رو تأیید کن، وگرنه ممکنه آلارم دیر یا نامنظم بزنه.
- توی تنظیمات گوشی (بسته به برند: شیائومی/سامسونگ و غیره) اپ رو از حالت **بهینه‌سازی باتری (Battery Optimization)** خارج کن و اجازه بده در پس‌زمینه اجرا بشه — وگرنه گوشی ممکنه اپ رو قبل از رسیدن زمان آلارم ببنده.
- برنامه هر روز ساعت ۰۰:۰۱ خودش رو دوباره زمان‌بندی می‌کنه، پس نیازی نیست هر روز اپ رو باز کنی؛ ولی بهتره حداقل یک‌بار در روز باز و بسته بشه تا مطمئن باشی.

## ساختار پروژه

```
app/src/main/java/com/amir/focusflow/
├── MainActivity.kt        صفحه‌ی اصلی (لیست برنامه + ساعت زنده)
├── AlarmActivity.kt        صفحه‌ی تمام‌صفحه‌ی آلارم + سؤال ریاضی
├── AlarmReceiver.kt        دریافت‌کننده‌ی زنگ از AlarmManager
├── AlarmScheduler.kt       زمان‌بندی همه‌ی آلارم‌های روز
├── BootReceiver.kt         بازسازی آلارم‌ها بعد از ریستارت گوشی / نیمه‌شب
├── ScheduleGenerator.kt    الگوریتم دقیق تولید برنامه (پورت از فایل HTML)
├── SessionStore.kt         ذخیره‌ی enabled/name ها (مثل localStorage)
├── MathBank.kt             ۱۰۰ سؤال ریاضی بانک اصلی
└── SessionAdapter.kt       نمایش لیست در RecyclerView
```

اگه خواستی زمان‌بندی رو تغییر بدی (مثلاً ساعت ناهار/شام یا طول بلوک مطالعه)، فقط کافیه فایل
`ScheduleGenerator.kt` رو ویرایش کنی.
